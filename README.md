# ReAct Agent: Гепард и Москва-река

Это реализация AI-агента, который умеет рассуждать, искать достоверную информацию и решать составные задачи.

В качестве демонстрации агент решает тестовую задачу:
> *"Сколько понадобится времени гепарду, чтобы пересечь Москву-реку по Большому Каменному мосту?"*

Проект демонстрирует подход **ReAct (Reasoning + Acting)** и протокол взаимодействия **A2A (Agent-to-Agent)**.

## Как это работает?

1.  **Рассуждает (Think):** Разбивает задачу на подзадачи ("Узнать длину моста", "Узнать скорость гепарда").
2.  **Действует (Act):** Ищет факты **WikipediaTool** или считает формулы **CalculatorTool**.
3.  **Наблюдает (Observe):** Анализирует данные.
4.  **Проверяет себя (Reflect):** Перед выдачей проверяет логику и физические единицы.

## Ключевые особенности

* **OpenRouter Integration:** позволяет переключаться между **GPT**, **Gemini** или **llama** (меняется .env).
  Для работы агента необходимо ввести свой API_KEY "OPENROUTER_API_KEY=..." в .env
* **Калькулятор:** LLM часто ошибаются в расчетах. Агент использует Python-калькулятор для вычислений.
* **Строгая типизация:** Type Hints и **Pydantic**.
* **A2A Протокол**
